# DEV.Rapha — Portfólio Pessoal

Site portfólio de **Raphael Castilho**, desenvolvedor Frontend com foco em QA e automação de testes.

## Acesso

> **[Ver online →](https://raphacastilho.github.io/portfolio)** *(em breve via GitHub Pages)*

## Funcionalidades

- Seção de apresentação com animação de digitação
- Timeline de formação e conhecimentos
- Seção de serviços oferecidos
- Galeria de projetos com filtros por categoria (Web Page, App Web, QA & Testing)
- Formulário de contato funcional via [FormSubmit](https://formsubmit.co/)
- Alternância de tema (dark/light) com persistência
- Suporte a dois idiomas: Português 🇧🇷 e Inglês 🇺🇸
